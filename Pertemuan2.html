Perancangan web adalah proses membangun dan menyusun halaman web agar dapat digunakan dan diakses dengan baik oleh pengguna. HTML (HyperText Markup Language) merupakan bahasa utama yang digunakan dalam perancangan web untuk membentuk struktur dan konten halaman.HTML bekerja dengan menggunakan tag untuk menampilkan berbagai elemen seperti teks, gambar, tabel, formulir, dan tautan. Setiap halaman web yang dibuat dengan HTML memiliki kerangka dasar yang terdiri dari bagian header, body, dan footer.Dalam proses perancangan web, HTML sering dikombinasikan dengan CSS untuk mendesain tampilan dan JavaScript untuk menambahkan interaktivitas. Dengan pemahaman dasar HTML, seseorang dapat membangun halaman web sederhana dan mengembangkannya menjadi lebih kompleks sesuai dengan kebutuhan.
